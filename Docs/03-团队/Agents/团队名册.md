## 维护规则

- 每次走 `Team` 路由或启用任何子代理后，必须更新本名册。
- 新 agent 或新昵称首次出现时，新增一行并创建对应成员档案。
- 同一 agent/nickname 再次参与任务时，更新最近任务和任务次数。
- 名册只写摘要，不粘贴完整工作记录、长命令输出或大段分析。
- 如果本轮没有启用子代理，但用户要求团队流程，应记录 `Codex 主线程` 的统筹工作。
