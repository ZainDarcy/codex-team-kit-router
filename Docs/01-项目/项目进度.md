# 项目进度

本文件是目标项目接入模板后的动态进度占位。模板源仓库不得在这里记录自身维护进度。

稳定规范见 [`项目规范.md`](./项目规范.md)，团队流程见 [`开发团队.md`](../03-团队/开发团队.md)。

## 使用说明

- 如果目标项目已有项目进度真相源，优先保留已有文件，不复制本文件内容。
- 如果目标项目没有项目进度文件，可使用本文件作为初始模板。
- 同类项目进度只能有一个真相源；其他位置只能建立索引，不复制正文。
- 本文件只记录目标项目进度，不记录本模板源仓库的维护历史。
